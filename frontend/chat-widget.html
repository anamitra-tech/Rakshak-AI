Prahari Assistant — embeddable chat widget

This file is a self-contained reference implementation of the floating chat bubble + slide-up quick-question/language panel for external-website integration with the /chat endpoint. Open this file directly in a browser to try it, or copy the #prahari-widget-root markup/CSS/JS block into your own page.

Before using on a real site: set CONFIG.apiBase and CONFIG.apiKey at the top of the script below. See the comment there — embedding the shared API key in browser JS is a known, disclosed temporary posture (matches webhook/app.py's current wide-open CORS, both flagged "must narrow/proxy before real launch" in API_SPEC.md §7.1.2) and should be replaced with a same-origin server-side proxy before a real production launch.

The API contract is unchanged: this widget only ever calls POST /chat {session_id, message} with header X-API-Key, exactly as documented in API_SPEC.md §7.1 — the language picker below never adds a language parameter to the request. It works purely by choosing which native-script text to send; /chat's own script-detection + translate bridge (Part 1 of this change) does the rest.
